What is CSS Relative Positioning?

  • CSS Relative Positioning is a method of positioning/moving elements in your website.
  • Positioned relative to "normal position"
  • Other elements act as if it is still there

Examples

position: relative;
top: 0px;
right: 0%;
left: 0px;
bottom: 0%;
